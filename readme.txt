# 🧠 OOP Abstraction & Polymorphism Practice (C++)

This repository contains simple C++ programs demonstrating **Abstraction** and **Runtime Polymorphism** using **pure virtual functions**.

---

## 📌 Concepts Covered

- Abstract Classes
- Pure Virtual Functions
- Method Overriding
- Runtime Polymorphism
- Base Class Pointers

---

## 💡 Key Idea:

> One interface, multiple implementations.

A base abstract class defines a contract, and derived classes implement their own behavior.

---

## 🚀 Programs Included:-

### 1. Shape System 🎨
- Classes: `Circle`, `Rectangle`, `Triangle`
- Function: `draw()`

---

### 2. Payment System 💳
- Classes: `UPI`, `CreditCard`, `PayPal`
- Function: `pay()`

---

### 3. Animal Sound System 🐾
- Classes: `Dog`, `Cat`, `Cow`
- Function: `sound()`

---

### 4. Employee System 👨‍💼
- Classes: `Developer`, `Manager`, `Tester`
- Function: `work()`

---

### 5. Vehicle System 🚗
- Classes: `Car`, `Bike`, `Bus`
- Function: `start()`

---

## 🧪 Learning Outcome

After practicing these examples, you will understand:

- How abstraction hides implementation details
- How polymorphism enables dynamic behavior
- Why base class pointers are important
- How real-world systems use OOP design
